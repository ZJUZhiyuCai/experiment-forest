# 硅基流动API配置指南

## 🚀 快速开始

### 1. 注册硅基流动账户
1. 访问 [硅基流动官网](https://siliconflow.cn)
2. 点击"注册"按钮，使用手机号或邮箱注册
3. 完成实名认证（需要身份证信息）

### 2. 获取API密钥
1. 登录后进入"控制台"
2. 在左侧菜单找到"API密钥"
3. 点击"创建密钥"
4. 复制生成的API密钥（格式：sk-xxxxxxxxxxxxxxxx）

### 3. 在系统中配置API

#### 配置步骤：
1. 进入系统设置页面
2. 找到"AI API配置"部分
3. 开启"使用自定义API"开关
4. 填写以下信息：

```
API端点: https://api.siliconflow.cn/v1/chat/completions
API密钥: sk-xxxxxxxxxxxxxxxx (你的密钥)
模型选择: qwen2.5-72b-instruct (推荐)
```

5. 点击"测试连接"验证配置
6. 测试成功后点击"保存设置"

## 🔧 常见问题解决

### 1. 连接失败 (网络问题)
**错误信息：** "Failed to fetch" 或 "网络连接失败"

**解决方案：**
- 检查网络连接是否正常
- 确认API端点地址正确：`https://api.siliconflow.cn/v1/chat/completions`
- 硅基流动支持国内直连，通常不需要代理

### 2. 认证失败 (401错误)
**错误信息：** "API密钥验证失败"

**解决方案：**
- 检查API密钥是否正确复制（包含sk-前缀）
- 确认密钥没有过期
- 检查账户余额是否充足
- 确认选择的模型有使用权限

### 3. 请求频率限制 (429错误)
**错误信息：** "请求频率限制"

**解决方案：**
- 等待几分钟后重试
- 检查是否超出了API调用频率限制
- 考虑升级账户套餐

### 4. 模型不存在 (404错误)
**错误信息：** "模型不存在"

**解决方案：**
- 检查模型名称是否正确
- 确认账户有该模型的使用权限
- 尝试使用推荐的 `qwen2.5-72b-instruct` 模型
